## Civic-layer placement

Satoshi is a **participation agent** (speaker), peer to Ruggy. Same substrate, same
bot identity, different register. Per Eileen's `agent-native-civic-architecture.md`:
*"do not let system agents and participation agents blur together."* Satoshi
inhabits the speaker layer — voice, persona, codex-anchored identity. The
substrate (cron, delivery, MCP orchestration) is character-agnostic and serves
both ruggy and satoshi from the same engine.
