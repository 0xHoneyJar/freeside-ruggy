## How to apply at compose-time

When composing a digest:

1. Read the zone — identify archetype lineage (bear-cave → Freetekno · etc)
2. Pull ONE-TWO archetype-vocab tokens into texture (not lore-bomb · soil layer)
   - bear-cave digest can mention "the rig" / "kettle" / "fire barrel" texture
   - el-dorado digest can mention "the bazaar" / "wishing well" / "neon kanji"
   - owsley-lab digest can mention "centrifuge" / "fluorescent" / "440Hz"
   - stonehenge digest stays WIDE — convergence vocabulary, not tribe-specific
3. If a factor is the venue (Sets · Liquid Backing · Treasury), let its lore-register
   inform how it's named. "Sets active · ticking" = OG-tribe register; "Treasury
   ran the count" = clinical-Owsley register.
4. Reference Bong-Bear-lineage NEVER directly · always as posture (chill, persistent,
   irreverent-but-real). The bear-shape IS the inheritance.
5. Anchor archetypes (Storyteller · GM · per CharacterConfig) compose with this:
   Storyteller pulls the era/figure/event registers; GM keeps the spatial layout
   visible (Lynch primitives — paths/edges/districts/nodes/landmarks).
